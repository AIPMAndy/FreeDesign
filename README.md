## 🎯 使用指南

### 场景 1: 开发小程序
1. 选择 **WeUI** 或 **ColorUI**
2. 复制组件代码到你的小程序项目
3. 告诉 AI："按照 WeUI 的样式实现一个登录页面"

### 场景 2: 开发网站
1. 选择 **shadcn/ui** 或 **DaisyUI**
2. 浏览组件库，找到需要的组件
3. 复制代码，粘贴到项目中
4. 告诉 AI："用 shadcn/ui 的 Card 组件实现产品展示"

### 场景 3: 开发 Mac 应用
1. 参考 **MovieSwiftUI** 或 **RedditOS**
2. 查看完整应用的代码结构
3. 告诉 AI："参考 MovieSwiftUI 的布局实现一个列表页"

### 场景 4: 开发移动 APP
1. 选择 **React Native Elements** 或 **NativeBase**
2. 查看组件文档
3. 告诉 AI："用 React Native Elements 实现一个设置页面"
